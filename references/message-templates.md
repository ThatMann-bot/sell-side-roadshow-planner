# Message Templates

Use natural WeChat-style Chinese. Keep messages short, specific, and copyable. Avoid formal email phrasing unless the user asks for it.

## Internal Sales Notice: Reschedule

```
[行程变更] 销售同事，机构A那场从下午改到上午10点，赵总那边已确认。

我这边顺序调成：
原来：机构B -> 机构C -> 机构A -> 机构D -> 机构E
现在：机构B -> 机构A -> 机构C -> 机构D -> 机构E

需要你帮忙确认：
✅ 机构A（赵总）：已确认，不用动
📞 机构C（王总）：从10:05改到11:30左右，需要跟对方确认
✅ 机构B、机构D、机构E：不受影响
```

## Buy-Side Notice: Reschedule

```
王总您好，跟您说一下周三拜访的时间调整。
原来约的10点出头我这边可能赶不过来了，您看11:30-12:15方便吗？
如果不合适，下午1:30左右也可以，我这边再配合您时间。
```

## Internal Sales Notice: Add Meeting

```
[新增拜访] 销售同事，机构X想加一场，我看了一下可以插在午饭后/机构Y之前。

当前建议顺序：
...

需要确认：
📞 机构X（X总）：确认具体地址和可接受时间
✅ 其他已约场次：暂不受影响

如果机构X只能固定在...，那会影响...，需要二选一。
```

## Buy-Side Notice: Add or Confirm Time

```
X总您好，我们周X在北京路演，想顺路拜访您一下。
我这边目前XX:XX-XX:XX比较合适，您看这个时间方便吗？
如果您那边更方便上午/下午，我也可以再协调一下路线。
```

## Internal Sales Notice: Cancel Meeting

```
[拜访取消] 销售同事，机构X这场先取消/改期了。

行程影响：
✅ 后面XX、XX时间不变
✅ 中间多出约X分钟 buffer，可以用于午饭/路上缓冲
📞 如果对方只是今天不方便，可以帮忙问一下是否改到明天上午/线上沟通。
```

## Buy-Side Notice: Cancel or Postpone

```
X总不好意思，今天路演安排这边有点调整，原定XX点的拜访可能需要先改期。
不想占用您临时等待时间，我这边跟销售再协调一个更稳妥的时间后同步您。
给您添麻烦了。
```

## Running Late Notice

```
X总您好，我这边上一场刚结束，路上预计还需要X分钟。
保守看大概XX:XX到，怕您等我先跟您同步一下。
如果您后面时间紧，我也可以到场后直接讲重点，尽量不影响您后续安排。
```
